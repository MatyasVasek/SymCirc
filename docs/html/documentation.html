class symcirc.AnalyseCircuit(netlist, analysis_type='DC', symbolic=True)¶

Main SymCirc class. When initialized it parses input netlist, and conducts the desired analysis which is then stored as a set of equations in the equation_matrix variable.

Parameters

  • netlist (str) – Loaded netlist file which contains the circuit description. If you intend to load from a file, use the utils.load_file() function.

  • analysis_type (str) – Analysis type identifier: “DC”, “AC”, “TF”, “tran”.

  • symbolic (bool) – False if you want your results evaluated with numerical values from the netlist.

Raises

ValueError – If the analysis_type argument is invalid.
