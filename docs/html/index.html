Welcome to SymCirc’s documentation!¶

SymCirc is a Python library for symbolic circuit analysis. It can do DC, AC and transient analysis of small signal circuits and output a human readable result.

Check out the Usage section for further information, including how to install the project.

Note

This project is my bachelor thesis and is under development.
